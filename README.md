# Joshua Owolabi - Portfolio Website

This is a simple portfolio website built with HTML, CSS, and JavaScript, based on the Bento Grid design.

## How to use

1. Open `index.html` in any web browser.
2. To customize the content further, edit `index.html`.
3. To change colors or layout, edit `styles.css`.

## Structure

- `index.html`: The main structure and content.
- `styles.css`: The styling, grid layout, and color theme.
- `script.js`: Simple interactions (smooth scrolling).

## Assets

- Fonts are loaded from Google Fonts (Manrope).
- Images use placeholders. Replace the `src` attribute of the `<img>` tag in `index.html` with your own photo.
